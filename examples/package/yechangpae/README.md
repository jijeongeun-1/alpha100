# 예비창업패키지 학습 예시 PDF

이 폴더에 PDF 파일을 넣으면 자동으로 학습 데이터로 사용됩니다.

## 파일 분류 기준 (파일명 키워드)

| 역할 | 파일명에 포함되어야 할 키워드 |
|------|-------------------------------|
| 사전보고서 | `과업지시서`, `사전`, `계획서`, `협약서` |
| 사후보고서(완성 예시) | `결과보고서`, `사후`, `완료보고` |
| 결과물 | `결과물`, `산출물`, `작업물` |

## 예시 파일명

- `01. 아이덴티티(하우뷰티), 앱 개발 과업지시서_ABBG.pdf` → 사전보고서
- `05. 아이덴티티(하우뷰티), 앱 개발 결과보고서_ABBG.pdf` → 사후보고서(완성 예시)
- `06. 아이덴티티(하우뷰티), 앱 개발 결과물_ABBG.pdf` → 결과물
